# 📚 Bibliothèque de CV - Baptiste Sandoz

Ce dépôt GitHub sert de coffre-fort et d'espace de travail pour l'ensemble de mes Curriculum Vitae. Il centralise mes versions terminées, mes brouillons et mes variantes (alternance, stage, projets spécifiques).

---

## 🛠️ Concept du dépôt
Contrairement à un CV classique, ce projet est géré comme un **projet logiciel** :
- **Versionnage** : Historique complet de mon parcours (chaque commit correspond à une mise à jour d'expérience ou de compétence).
- **Format "Source"** : Les CV sont codés en **HTML/CSS** pour une flexibilité totale.
- **Organisation** : Centralisation des assets (photos, logos d'entreprises/écoles) et du code.

## 📂 Structure du dépôt
- `/PROJET-ACTUEL` : La version la plus récente de mon CV (HTML/CSS).
- `/ARCHIVES` : (Optionnel) Mes anciennes versions pour garder une trace de mon évolution.
- `/ASSETS` : Ma photo de profil et le logo de l'**École IT**.

## 🚀 Stack Technique
- **Design** : HTML5 / CSS3 (Flexbox & Print Media Queries).
- **Charte Graphique** : Identité visuelle École IT (#B2175B).
- **Outils** : VS Code pour l'édition, Git pour le suivi.
